0. Overview

A collaborative design project focused on creating a digital embroidery tool that allows users to explore, customize, and visualize stitch patterns in creative and meaningful ways. We aimed to support accessibility, personalization, and community engagement through thoughtful feature planning.

1. My Role

Storyboard Creation · Diagram Design · UI Contribution

I created user-focused storyboards, developed system diagrams, and contributed to the UI prototype in Figma—focusing on specific features that addressed common user challenges.

2. Design Decision

To understand user needs, I began by illustrating common frustrations in the form of storyboards (Figures 1–2), such as not knowing which fabric or pattern to choose, difficulty visualizing end results, and limitations with basic input tools.

These scenarios shaped our design priorities. I then supported the translation of these ideas into functional diagrams and contributed to UI development in Figma to prototype the tool’s core interactions.

3. Result

We created an interactive prototype in Figma that demonstrates how the tool could support users in selecting fabrics, browsing shared patterns, previewing outputs on objects, and applying pattern brushes. While not a functioning app, the prototype offers a realistic simulation of the user experience and design logic.
