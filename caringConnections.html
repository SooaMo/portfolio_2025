Caring Connections

Academic • 2024 Nov-Dec

Team: Sooa Mo, Hao N., Kellystin N., Vincent S., Winnie S.

Project Type: • Board Game Design • Tabletop Simulator Prototype

0. Overview

A cooperative board game that encourages empathy, care, and reflection through storytelling and shared decision-making. Designed to promote meaningful conversation in a safe and playful format.

Watch Trailer Video View Rulebook Visit Website View Report (PDF) Steam Board Game link

1. My Role

I designed the visual elements of all cards and tokens, created most game boards, and co-developed the rulebook layout. I also supported team coordination, recorded playtests, and edited the trailer video.

2. Design Highlights

• Game Components

I created the card visuals, icons, and illustrated tokens (gold, silver, pills) to support clarity and emotional tone. Also, Designed all boards except the counterboard, maintaining a unified tone and soft, calming visuals.

Fig. 1: Final card and token set used in the game
• Rulebook Design

Collaborated on transforming the draft into a polished, accessible rulebook using graphic layout and visual hierarchy.

Fig. 2: Rulebook spread showing clear icons and structure
• Playtesting & Media

Recorded most test sessions and edited the final trailer to document the game experience.

Fig. 3: Game play screenshot + playtesting document record

3. Takeaway

This project taught me how to translate abstract concepts like care and vulnerability into visual systems, and how thoughtful collaboration can elevate a social experience through design.
